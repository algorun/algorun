Web API

This algorithm can be run via the web interface above or via the following web API

HTTP POST /v1/run

    Parameters

  • input: the input for the algorithm in its accepted format. See: sample-input-format

    Results

  • output: the result of the computation in the following format. See: sample-output-format

HTTP POST /v1/config

    Parameters

  • JSON Object
    key: parameter name
    value: the new value of parameter
    You can change more than one parameter per API call

    Results

  • Success or failure message
